This entry describes compiling and running rust gstreamer examples.

Install Pre-Requisites

apt-get install libgstreamer1.0-dev libgstreamer-plugins-base1.0-dev \
      gstreamer1.0-plugins-base gstreamer1.0-plugins-good \
      gstreamer1.0-plugins-bad gstreamer1.0-plugins-ugly \
	  gstreamer1.0-libav libgstrtspserver-1.0-dev libges-1.0-dev

Clone Repository

git clone --recursive https://gitlab.freedesktop.org/gstreamer/gstreamer-rs.git

Compile Package

For my setup, I noticed I was missing a rust component and got the following error.

python3 generator.py
error: no such subcommand: `fmt`

Install rustfmt

cargo install rustfmt
python3 generator.py
cargo build

Run Example

cargo run --bin basic-tutorial-1
