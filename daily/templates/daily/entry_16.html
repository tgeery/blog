This entry describes installing Houdini (visual effects software) on Ubuntu.

Procedure

Download the installation package from Side FX downloads page.

Extract and begin installation script

$ tar -xzvf houdini-18.0.287-linux_x86_64_gcc6.3.tar.gz
$ cd houdini-18.0.287-linux_x86_64_gcc6.3/
$ sudo ./houdini.install

Complete license agreement.

Select kit to install and enter the default 'F'.

Wait for installation completion.
