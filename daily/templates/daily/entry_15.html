This entry describes basic usage of Highlight.js to dislplay code snippets within HTML <pre><code> tags using syntax highlighting.

Pre-requisite

This example will be used within a Django project that is including bootstrap CSS.

Simple non-highlight.js example

Example referenced from https://themes.gohugo.io/hugo-book/.

<html>
  <head>
    <style>
      #terminal { 
        color:#f8f8f2;
        background-color:#272822;
        -moz-tab-size:4;
        -o-tab-size:4;
        tab-size:4;
      }
    </style>
  </head>
  <body>
    <pre id="terminal">$ python3 -m pip install psycopg2-binary</pre>
  </body>
</html>

Highlight.js Example

<html>
  <head>
    <link rel="stylesheet" href="//cdn.jsdelivr.net/gh/highlightjs/cdn-release@9.17.1/build/styles/default.min.css">
    <script src="//cdn.jsdelivr.net/gh/highlightjs/cdn-release@9.17.1/build/highlight.min.js"></script>
    <script>hljs.initHighlightingOnLoad();</script>
  </head>
  <body>
    <pre><code class="bash">#!/bin/bash
      echo "hello world"
    </code></pre>
  </body>
</html>
