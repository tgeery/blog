This entry describes a quick setup of Travis-CI using YAML to implement Django tests and display Github badges.

Install Pre-requisites

Setup an account and project with github.com, as well as an account with travis-ci.com. Next, follow travis-ci instructions to activate Github(or Bitbucket) projects.

Setup Project

Edit README.md

[![Build Status](https://travis-ci.com/username/project.svg?branch=master)](https://travis-ci.com/username/project)

Create travis.yml

language: python

python:
    - "3.6"

env:
    - DJANGO_VERSION=3.0.6

install:
    - pip3 install -r requirements.txt

script:
    - python3 manage.py test
