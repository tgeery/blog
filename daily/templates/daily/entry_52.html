This entry describes the setup for a hello world c++ app using vs code on Ubuntu following this tutorial.

Install VS Code

snap install code

Create C++ App

mkdir hello
cd hello

Create hello.cpp file.

#include 

int main() {
    std::cout << "Hello World!\n";
    return 0;
}

Install VS Code Extensions

code .

Use shortcut to install extensions: ctrl+shift+x
install extension:
    ms-vscode.cpptools
    acharluk.easy-cpp-projects

Build and Run Project

Select cpp file
Terminal->Configure Tasks...
Select 'C/C++:g++ build active file'

Select cpp file
Terminal->Run Build Tasks...
Select 'C/C++:g++ build active file'

Select cpp file
Run->Run Without Debugging

Verify App

Verify terminal prints: Hello World!
