This entry describes the basic setup for an Nvidia Jetson Nano. The Nano is only connected using micro USB and uses a micro SD; however a keyboard, mouse, and HDMI were used to configure initial board bring up.

https://developer.nvidia.com/jetson-nano-sd-card-image-r3231
https://www.balena.io/etcher/
cd ~/Downloads/
unzip balena-etcher-electron-1.5.71-linux-x64.zip
./balenaEtcher-1.5.71-x64.AppImage

select storage device

select flash

May require a reboot and then verify ubuntu bootup and perform initial configuration.
