This entry describes a simple setup of jekyll as a blog site and an initial page.

Install Pre-requisites

sudo apt-get install ruby ruby-dev make build-essential
sudo gem install jekyll bundler

Setup Project

jekyll new site
tree site/
site/
├── 404.html
├── about.markdown
├── _config.yml
├── Gemfile
├── Gemfile.lock
├── index.markdown
└── _posts
    └── 2020-05-26-welcome-to-jekyll.markdown

1 directory, 7 files
cd site
jekyll serve --host=127.0.0.1

Add an Entry to the Project

vim _posts/2020-05-26-entry-1.md
---
layout: post
title: "Entry 1"
---
This is my first entry.
jekyll serve --host=127.0.0.1
