This entry describes a simple way to use the python opencv library to take a screenshot of your desktop.

Install Pre-requisites

sudo apt-get install python3-tk python3-dev scrot
pip3 install pyautogui opencv-python

Run Python Code

python3
Python 3.7.5 (default, Apr 19 2020, 20:18:17) 
[GCC 9.2.1 20191008] on linux
Type "help", "copyright", "credits" or "license" for more information.
>>> import pyautogui, cv2, numpy as np
>>> cv2.imwrite("test.png", cv2.cvtColor(np.array(pyautogui.screenshot()), cv2.COLOR_RGB2BGR))
True
>>> quit()

View Result

eog test.png
