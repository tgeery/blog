This entry describes building QtGStreamer module from source and running the player example with a sample mp4 file.

Install Prerequisites

sudo apt-get install libboost-all-dev qt4-qmake libqt4-dev doxygen

Clone and Build Repo

git clone --recursive https://anongit.freedesktop.org/git/gstreamer/qt-gstreamer.git
cd qt-gstreamer/
make

Download sample file

Player application can play audio or video. Use a sample video file.

Run Player Example

examples/player/player

Select video file. Press play.
