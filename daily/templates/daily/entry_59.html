This entry describes setting up VS Code to ssh to a remote device and modify directory files.

Install Extension

Select Extensions Tab

Search 'remote-ssh' and install

Add Device to Remote List

Select Remote Explorer Tab

Click +, and add <user>@<ip address>

Click the icon next to ip address and then type the password

Open Directory

Select File->Open Folder (go to project)

Select Terminal->New Terminal (run code)
