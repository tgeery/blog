This is a reference to networking Open Systems Interconnection(OSI) model layers as described on www.computernetworkingnotes.com and www.plixer.com/.

OSI Layers

physical layer (e.g. Ethernet, USB, Bluetooth)
data link layer (e.g. MAC, PPP, 802.3, 802.11)
network layer (e.g. IP, router, packet)
transport layer (e.g. segmentation, TCP, UDP)
session layer (e.g. syn/ack, full-duplex, half-duplex, RPC, NFS)
presentation layer (e.g. XML, ASCII, PNG)
application layer (e.g. HTTP, SSH, FTP, TFTP)
